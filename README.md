# Full Stack Application

This is a full-stack application with Vite React frontend and Node.js backend.

## Project Structure
- `client/` - Frontend (Vite + React)
- `server/` - Backend (Node.js + Express)

## Getting Started

### Frontend
```bash
cd client
npm install
npm run dev
```

### Backend
```bash
cd server
npm install
npm run dev
```
